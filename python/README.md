# Kuro-Roku ML Sidecar

Python ML processing sidecar for video analysis and transcription.

## Components

- **vlm.py** - Vision-Language Model inference (Qwen3-VL)
- **transcription.py** - Whisper transcription
- **embeddings.py** - SigLIP2/embedding generation
- **server.py** - FastAPI server or stdio interface

## Setup

```bash
cd python
pip install -e .
```
